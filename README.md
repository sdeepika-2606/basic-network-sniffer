# Basic Network Sniffer

## Description
This project implements a basic network sniffer using Python and the Scapy library.
It captures live network packets and displays source IP, destination IP,
protocol details, ports, and payload information.

## Tools Used
- Python 3
- Scapy
- Kali Linux
